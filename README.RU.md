# Numpad Plus
Запуск приложений и изменение громкости горячие клавиши. Поддерживается до 12 кнопок.

## Горячие клавиши
По умолчанию используются `Numpad` клавиши вместе с кнопкой `ALT`. Кнопки можно изменить в конфигурационном файле.



По умолчанию изменение громкости на клавиши `ALT + Numpad Плюс` и `ALT + Numpad Минус`



При необходимости можно отключить горячие клавиши, через контекстное меню в области уведомлений (tray).

## Настройка
Введите в конфигурационный файл `Config.ini` названия и пути приложений. Также при необходимости можно указать параметры запуска для приложений.



Можно задать уровень громкости системы по умолчанию, параметр `StartupVolume`, применяться будет при запуске программы. 



Для автозапуска добавьте ярлык программы `NumPadPlus.exe` в автозагрузку - Windows `%AppData%\Microsoft\Windows\Start Menu\Programs\Startup`.
